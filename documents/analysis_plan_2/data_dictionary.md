| 列名 | データ型 | 説明 |
| --- | --- | --- |
| ID | 整数 | 被験者識別子 |
| visit_date | 文字列 | 受診日（YYYY-MM-DD） |
| age | 整数 | 受診時の年齢（歳） |

| height | 数値 | 身長（cm） |
| smoke | 整数（0: 非喫煙, 1: 喫煙） | 喫煙状況（0: 非喫煙, 1: 喫煙） |

| CRE | 数値 | クレアチニン値（mg/dL） |
